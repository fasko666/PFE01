# Panda — Frontend

React frontend for the Panda freelance marketplace platform.

## Tech Stack

- **React** — UI framework
- **Vite** — Build tool
- **Tailwind CSS** — Styling
- **Zustand** — State management
- **Axios** — API client

## Features

- Landing page with 3D animations
- Authentication (Login, Register, Google OAuth, Onboarding)
- Freelancer and Client dashboards
- Job marketplace and proposals
- Real-time messaging
- Payments
- AI assistant
- Admin dashboard

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```
